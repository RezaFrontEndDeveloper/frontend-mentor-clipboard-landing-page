# Clipboard Landing Page

A responsive landing page built with HTML and CSS.

## Overview

This project is a solution to a landing page challenge. The goal was to practice responsive layouts using Flexbox and CSS.

## Features

- Responsive Design
- Flexbox Layout
- Mobile Friendly
- Semantic HTML

## Built With

- HTML5
- CSS3
- Flexbox
- Media Queries
